# Cline Prompt Template - Weekly Retrospective

## Prompt

```text
You are my learning coach. Based on this week's artifacts:
- workshop result
- assignment result
- review comments
- failed tests/errors

Provide:
1) Top 3 skill gains
2) Top 3 skill gaps
3) Root cause analysis for mistakes
4) Next-week focus plan (max 5 actions)
5) One deliberate practice exercise
```
