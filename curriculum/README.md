# Curriculum Index

> 使用方式：每週先讀完該章 `章節導讀 + 核心知識`，再做 `Code-Along（逐步照做）`。  
> 每章都已內建 0 基礎知識、圖表（ASCII + Mermaid）、可貼上程式碼、驗收條件、卡關排查、Wrong vs Right 程式碼對照、每週客製章節練習與交付證據清單。
> 所有週次的 Mermaid 原始圖檔請見 [../assets/diagrams/README.md](../assets/diagrams/README.md)。

## 自學者每日流程（固定照做）

1. 先讀本週章節前半（導讀、詞彙、核心知識）。
2. 再做本週 `Code-Along` 的 Step 0 到 Step 4。
3. 再做本章 `章節練習（不看答案先做）`。
4. 卡住超過 10 分鐘，先看本章 `常見錯誤與排查` 與 `提示（卡住再看）`，再用 Cline 問最小重現。
5. 每天留下證據（筆記、截圖、測試結果、diff）。
