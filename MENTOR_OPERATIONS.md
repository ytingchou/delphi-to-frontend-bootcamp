# Mentor Operations Guide

## Weekly Review Cadence

1. Monday: scope lock and risk preview
2. Wednesday: workshop checkpoint review
3. Friday: assignment grading + retrospective

## Review Stack

- Workshop acceptance criteria
- Assignment expected solution shape
- Rubric scoring
- Cline response review checklist

## Intervention Rules

- If rubric < 80 for 2 consecutive weeks: reduce scope, increase guided labs.
- If security hard gate fails once (Week 13+): mandatory remediation before next week.
- If Cline usage quality < 3/5: force prompt-quality coaching session.

## Agent-style Review Roles

- Architecture reviewer: boundary decisions (RSC/Client/BFF)
- Security reviewer: auth, token, CORS, role checks
- QA reviewer: lint, tests, failure paths
- Delivery reviewer: PR quality, risk notes, rollback readiness

## Graduation Readiness Interview

- 15 min architecture defense
- 10 min incident-debug simulation
- 10 min code review simulation with Cline suggestion critique
