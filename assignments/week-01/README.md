# Assignment Week 01 - Web Runtime Report

## Task

撰寫一份「前端請求生命週期」報告並附一個最小 demo，展示 browser -> server -> API 的實際互動。

## Required Output

- `runtime-report.md`（至少 800 字）
- demo 頁面（可觸發成功與失敗請求）
- DevTools 截圖 3 張（成功、4xx、5xx）

## Expected Solution Shape

- 報告至少含：request path、status code 解讀、header/cookie 基礎。
- demo 含 loading/error UI。
- 能重現且可解釋錯誤。

## Cline Journey

1. 用 Cline 拆解報告大綱。
2. 用 Cline 檢查解釋是否有概念錯誤。
3. 人工核對與 MDN/官方資料一致性。

## Submission Checklist

- [ ] 有可執行 demo
- [ ] 報告結構完整
- [ ] Cline 審查紀錄
