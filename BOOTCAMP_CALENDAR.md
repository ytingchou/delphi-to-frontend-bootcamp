# 16-Week Bootcamp Calendar

## Milestone Gates

- Gate A (End of Week 04): Web/JS foundation pass
- Gate B (End of Week 08): TypeScript + React baseline pass
- Gate C (End of Week 12): Next.js/RSC/BFF maintenance-ready
- Gate D (End of Week 16): Auth + quality + capstone delivery pass

## Weekly Time Allocation (20+ hr)

- 30% concept learning
- 40% workshop implementation
- 20% assignment + tests
- 10% Cline review + retrospective

## Graduation Standard

- No failed hard gates from rubrics
- Capstone accepted
- Demonstrated independent issue triage with Cline + human review
