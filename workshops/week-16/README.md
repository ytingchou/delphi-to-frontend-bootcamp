# Workshop Week 16 - Capstone Delivery Sprint

## Problem

以企業後台場景完成最終交付衝刺，整合 RSC、BFF、OIDC、測試與 release artifacts。

## Scope

- Scope freeze（必做 vs 可選）。
- 交付主流程：登入->儀表板->受保護 API。
- 完成 lint/test/PR narrative。

## Constraints

- 時間箱（timebox）內交付。
- 不可為了趕進度移除安全檢查。
- PR 文件必須包含風險與回滾。

## Deliverables

- 可演示成品
- `architecture-decision-log.md`
- `release-risk-plan.md`
- Cline 完整對話摘要

## Cline Task

- 用 Cline 協助拆解 scope 與風險。
- 請 Cline 進行最終 PR reviewer 模擬。
- 人工標記 Cline 錯誤建議與修正證據。

## Acceptance Criteria

- 全部必做需求達成。
- 品質閘通過。
- 技術 demo 可完整辯護設計決策。
